NAME

       cos - cosine function


SYNOPSIS

       #include <math.h>

       double cos(double x);


DESCRIPTION

       The  cos()  function  returns  the cosine of x, where x is
       given in radians.


RETURN VALUE

       The cos() function returns a value between -1 and 1.


CONFORMING TO

       SVID 3, POSIX, BSD 4.3, ISO 9899
