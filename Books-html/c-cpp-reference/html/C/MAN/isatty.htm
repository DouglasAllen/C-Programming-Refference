NAME

       isatty - does this descriptor refer to a terminal


SYNOPSIS

       #include <unistd.h>

       int isatty ( int desc );


DESCRIPTION

       returns  1  if  desc  is an open descriptor connected to a
       terminal and 0 else.


CONFORMING TO

       SVID, AT&T, X/OPEN, BSD 4.3
