Ruby/VTE

Ruby/VTE is a Ruby binding of VTE.

Requirements 

    Ruby:               http://www.ruby-lang.org/
    VTE [*]:            http://www.gnome.org/
    Ruby/GTK2:          http://ruby-gnome2.sourceforge.net/

    [*]: 0.12.1 or later is required.

Install 

    0. install ruby and VTE.
    1. ruby extconf.rb
    2. make
    3. su
    4. make install
