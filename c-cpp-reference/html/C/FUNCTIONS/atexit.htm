ATEXIT function

atexit regesters a function to be executed when the program terminates normally. 


	Library:   stdlib.h

	Prototype: int atexit(void (*function)(void));

	Syntax:    void End(void);
        
                   main()
                   {
                     atexit(End);
                   }

                   void End(void)
                   {
                     printf("The End\n");
                   }
