NAME

       tan - tangent function


SYNOPSIS

       #include <math.h>

       double tan(double x);


DESCRIPTION

       The  tan()  function  returns the tangent of x, where x is
       given in radians.


CONFORMING TO

       SVID 3, POSIX, BSD 4.3, ISO 9899
