NAME

       ceil - smallest integral value not less than x


SYNOPSIS

       #include <math.h>

       double ceil (double x);


DESCRIPTION

       The  ceil() function rounds x upwards to the nearest inte-
       ger, returning that value as a double.


CONFORMING TO

       SVID 3, POSIX, BSD 4.3, ISO 9899
